Copywriter Cat helps WordPress designers collect, draft, and approve on-brand website copy with clients—right in the page layout.

== Description ==

MVP scaffolding.

== Installation ==

1. Upload the plugin ZIP.
2. Activate.
3. In the block editor, insert “Copy Slot (Copywrite Cat)”.
